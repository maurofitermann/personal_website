hat_man

hat_man es un bot para mi servidor personal de Discord. Programado en Javascript, el bot corre en Node,js y se apoya principalmente en los módulos Discord.js para interactuar con la app y Mongoose.js para interactuar con una simple base de datos NoSQL.


Siendo un proyecto personal de tiempo parcial, hat_man es un utility bot sin enfoque. Hasta ahora lo he usado para:

  • Aprender a correr scripts en runtime, en este caso node.js
  • Aprender a ejecutar HTTP requests, interactuar con APIs
  • Utilizar API keys para autorizar mi acceso
  • Interpretar y utilizar contenido JSON
  • Mejorar mi entendimiento de los algoritmos y scripts basados en eventos
  • Aprender a crear, conectar, leer y escribir sobre bases de datos NoSQL en MongoDB utilizando Mognoose.js

Hablemos!
